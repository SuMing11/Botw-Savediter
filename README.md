### 关于 savediter.sh 这个脚本

这个是一个修改存档的辅助脚本，配置好后，可以自动创建存档备份，打开存档目录&修改页面。
提供了清理配置文件和存档备份的功能。这个脚本要求你保持 mlc01 文件夹在默认位置。
